# Coin Clicker

ブラウザで遊べる、シンプルなクリッカーゲームです。  
ボタンを押してコインを集め、アップグレードや建物を買って、少しずつ大きく育てていきます。

## どんなゲーム？

`Coin Clicker` は、クリックでコインを集めながら自動収入を増やしていくゲームです。

できること:

- ボタンを押してコインを集める
- コインを使って `クリック強化` を買う
- `おてつだいネコ` を増やして自動でコインを集める
- 条件を満たすと新しい建物を開放する
- `節目ボーナス` でコインの増え方を大きくする
- オート購入でゲームの進み方を自動化する

## 遊び方

1. `コインを集める` ボタンを押してコインを増やします。
2. たまに `ラッキー！` が出ると、その回は多めにコインがもらえます。
3. コインがたまったら、`クリック強化` や `おてつだいネコ` を買います。
4. `おてつだいネコ` を増やすと、毎秒コインが入るようになります。
5. 条件を満たすと、`ネコカフェ` などの建物が開放されます。
6. 建物を増やすと自動収入が大きくなり、さらに次の建物を目指せます。
7. 必要ならオート購入をオンにして、自動でアップグレードを進められます。
